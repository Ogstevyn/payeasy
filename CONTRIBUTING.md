# Contributing to PayEasy

Thank you for your interest in contributing to PayEasy. This guide explains how to get started.

## Ways to Contribute

- Code: Frontend features, smart contracts, bug fixes
- Design: UI/UX improvements, mockups, branding
- Documentation: Tutorials, guides, code comments
- Testing: Write tests, report bugs, QA
- Community: Answer questions, help newcomers

## Getting Started

### 1. Fork and Clone

```bash
# Fork the repo on GitHub, then clone your fork
git clone https://github.com/YOUR_USERNAME/payeasy.git
cd payeasy

# Add upstream remote
git remote add upstream https://github.com/Ogstevyn/payeasy.git
